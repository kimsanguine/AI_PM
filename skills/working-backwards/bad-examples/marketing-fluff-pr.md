# Bad 예시 — 마케팅 광고가 되어버린 PR

## 무엇이 문제인가

- 형용사 광고("혁신적", "업계 최고", "seamless")로 도배 → 무엇이 달라지는지 안 보임
- 근거 없는 수치를 사실처럼 단정
- 가상 고객 인용이 칭찬일 뿐, 어떤 진전이 있었는지 없음
- 내부 FAQ 가 통과를 전제 → 반대 의견·실패 조건 부재
- 검증 신호 없음 → "만들 가치" 를 끝내 검증하지 않음

## 출력 (이렇게 하지 말 것)

**헤드라인**: 업계를 뒤흔들 혁신적 AI 다이제스트, 드디어 출시!

**본문**: 우리의 혁신적이고 seamless 한 솔루션은 생산성을 3배 높이고 모든 문제를 해결합니다. 업계 최고의 기술로 누구나 손쉽게 사용할 수 있습니다.

> "정말 대단해요! 최고예요!" — 만족한 고객

**내부 FAQ**: Q. 잘 될까요? A. 네, 분명히 성공할 겁니다.

## 왜 나쁜가

- "혁신적/seamless/업계 최고" — 측정 불가. PR 은 고객 언어의 구체적 진전("월요일 취합 1시간 → 5분")으로 써야 함.
- "생산성 3배" — 근거 없는 수치를 사실로 단정. 근거 없으면 [가정] 이어야 함.
- "정말 대단해요" — 무엇이 어떻게 달라졌는지 0. 가상 인용은 before→after 를 말해야 함.
- "분명히 성공할 겁니다" — 내부 FAQ 의 목적(가장 불편한 질문, 실패 조건)을 회피. Working Backwards 의 핵심은 통과가 아니라 사전 검증.
- 검증 신호가 없어 "이 가정이 틀리면 안 만든다" 는 결정 회로가 비어 있음 → PR-FAQ 가 아니라 그냥 출시 광고.
