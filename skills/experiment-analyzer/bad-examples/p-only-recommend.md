# Bad 예시 — p-value 만 보고 ship 권고

## 무엇이 문제인가
- p<0.05 인데 effect size 가 사실상 0 (+0.02pp) 에서 ship 권고
- 가드레일 메트릭(환불율 +1.3pp 악화) 무시
- 표본 기간 2일 (novelty 효과 가능성) 무시
- 정성 의견 ("느낌이 좋다") 추가
- segment 편향 미검토

## 출력 (이렇게 하지 말 것)

p-value 가 0.04 로 유의하다. 주 메트릭이 통계적으로 증가했다. 느낌도 좋다.
**ship 권고.**

## 왜 나쁜가
- p=0.04, Δ=+0.02pp → "통계적 유의" 와 "비즈니스 임팩트" 가 따로 — ship 가치 없음
- 환불율 +1.3pp → 가드레일 위반 → ship 권고 자체가 부적합
- 2일 데이터 → novelty 효과 분리 불가, 추가 관측 필요
- "느낌도 좋다" → 정량 결과의 신뢰를 깎는 추가 의견
- segment 검토 없음 → 한 segment 가 결과를 끌고 갈 위험 (Simpson)

## 올바른 권고
**extend** (1주 추가 관측) → 가드레일 회복 안 되면 **kill**.
