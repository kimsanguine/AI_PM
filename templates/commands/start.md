# /start — 인터랙티브 온보딩 코스 인덱스

## Task

이 코스를 "읽는 문서"가 아니라 "Claude Code 안에서 직접 해보는 단계별 체험"으로 진행한다.
학습자의 레벨을 먼저 확인한 뒤, 알맞은 트랙(`/start-1`·`/start-2`·`/start-3`)으로 안내한다.

## Scope

- 이 커맨드는 안내자(가이드) 역할만 한다. 실제 학습 단계는 각 트랙 커맨드가 담당한다.
- 학습자를 대신해 과제를 끝내주지 않는다. 학습자가 직접 입력하고 실행하도록 유도한다.
- 처음 온 학습자에게 코스 전체 지도와 진행 방식을 먼저 설명한다.

## Length

- 인사 + 레벨 질문 + 트랙 안내까지 600자 이내. 짧고 친근하게.

## Format

먼저 다음을 그대로 안내한다.

```text
환영합니다. 이 코스는 Claude Code 안에서 AI_PM 을 단계별로 직접 체험하는 방식입니다.
읽기만 하지 않고, 매 단계 작은 과제를 직접 입력하고 실행해 봅니다.

진행 방식
- 한 번에 한 단계씩 진행합니다.
- 각 단계는 [읽을 챕터] → [직접 해볼 과제] → [30초 검증] 순서입니다.
- 다음 단계로 넘어가기 전에 항상 확인을 받습니다.
```

그다음 레벨을 묻는다. (`AskUserQuestion` 사용 가능)

```text
지금 어느 단계에 가까운가요?
- J (Junior): Claude Code 를 거의 처음 씁니다.
- P (Practitioner): 설치는 했고 기본 대화는 해봤습니다.
- L (Lead): 스킬·MCP·워크플로우를 더 깊게 쓰고 싶습니다.
```

선택에 따라 분기한다.

| 선택 | 다음 커맨드 | 다루는 범위 |
|---|---|---|
| J | `/start-1` | Part1~2 입문 — 설치·도구 지도·CLAUDE.md |
| P | `/start-2` | Part2~3 — 스킬·슬래시 커맨드·MCP 활용 |
| L | `/start-3` | Part4~6 — Discovery → Definition → Delivery 워크플로 관통 |

처음이라 잘 모르겠다고 하면 `/start-1` 부터 권한다.
마지막으로 "준비되면 위 커맨드 중 하나를 입력해 주세요" 라고 안내하고 멈춘다.
