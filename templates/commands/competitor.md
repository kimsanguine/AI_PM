# /competitor — 경쟁사 구조화 분석

## Task
지정된 경쟁사 3–5곳을 동일한 축으로 비교한 **구조화된 분석 표 + 배틀카드 초안**을 출력한다.

## Scope
- 정량/공개 자료 위주(가격, 기능, 리뷰 별점, 사용자 수). 추측·내부 정보 사용 금지.
- 본 레포 `samples/competitor-data.json` 또는 사용자가 첨부한 자료만 근거.
- 출력은 비교 표 + 배틀카드 초안까지. 실제 GTM 전략 변경은 하지 않는다.

## Length
- 비교 표 5–8행, 3–5열. 배틀카드 한 경쟁사당 ≤ 300자.
- 전체 1,500–3,000자.

## Format
```markdown
## 비교 축
| 축 | 우리 | 경쟁사 A | 경쟁사 B | 경쟁사 C |
|---|---|---|---|---|
| Target segment | ... | ... | ... | ... |
| Core feature   | ... | ... | ... | ... |
| Price          | ... | ... | ... | ... |
| Weakness       | ... | ... | ... | ... |

## 배틀카드 — 경쟁사 A
- **그들의 강점**: <한 줄>
- **그들의 약점**: <한 줄>
- **우리의 differentiator**: <한 줄>
- **그들이 우리에게 할 공격**: <한 줄> → **우리의 응답**: <한 줄>

## 한계
- 근거 자료의 freshness / 누락된 경쟁사 / 추가 조사 필요 항목
```

## Examples

<example>
<good>
| 축 | 우리 | A | B |
|---|---|---|---|
| Price | $29/mo | $99/mo | Free + $/seat |
| Target | mid-market PM | enterprise | startup |
| Weakness | 자동화 부족 | 가격 진입장벽 | 데이터 한계 |
</good>
<bad>
A 는 좋고 B 는 별로다. 우리가 더 낫다.
</bad>
</example>
