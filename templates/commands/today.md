# /today — 오늘의 PM 브리핑

오늘의 업무 브리핑을 생성합니다.

## 수행 단계

1. Linear에서 오늘 Due인 이슈와 In Progress 이슈를 가져옵니다
2. Slack #product, #engineering 채널의 어제 핵심 논의를 요약합니다
3. 오늘의 캘린더 일정을 확인합니다
4. 다음 형식으로 브리핑을 생성합니다:

```
## 📅 오늘의 브리핑 — [날짜]

### 🔴 긴급 (오늘 반드시)
- [Due today 이슈 목록]

### 🟡 진행 중
- [In Progress 이슈 요약]

### 💬 어제 핵심 논의
- [Slack 요약 3줄]

### 📆 오늘 일정
- [미팅 목록 + 준비 사항]

### 🎯 오늘의 포커스
- [가장 중요한 1가지 결정/작업]
```

5. `status/` 폴더에 `[날짜]-daily.md`로 저장합니다
